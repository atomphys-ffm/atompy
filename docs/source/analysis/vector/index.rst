======
Vector
======

With the :code:`Vector` class, :code:`atompy` offers some linear algebra
operations for vectors.

The underlying data structure is a NumPy `ndarray 
<https://numpy.org/doc/stable/reference/generated/numpy.ndarray.html>`_.
